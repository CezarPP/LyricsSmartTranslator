Abstract

The purpose of this paper is to describe the fornt-end part of project "LYRIT" that our team made. It lists the main requiremets, functionalities and various cases of user interaction. It also presents the tehnologies used and the motivation behind choosing them.

Introduction

Liryt is a website for reading, writing and commenting on poems tranlations in different languages. It is based on Project 6 (Poem Smart Translator), but instead of poems (with the agreement from our seminar teacher) it uses song lyrics instead of poems. The website offers the user the abbility to make an account and then add new songs, add translations, comment, share and make addnotations on the side of every song. It also displays different charts and rankings with the most popular translations or the most active users. Moreover, the newest translations are available via a RSS feed. In this first part of the project only the front-end was created.

Website features

Liryt is a complex website that offers users a lot of useful functionalities that help them learn about new songs and make contributing with new translations or songs very easy.

The most important elements of the website are:

  • Navbar
  • Main page
  • Search page
  • Song page
  • Add song page
  • Add translation page
  • Login page
  • Profile page
  • Other pages

Navbar

The website navbar remains consistent across all pages, featuring a straightforward menu that facilitates simple navigation for the user. The menu contains buttons that lead to different parts of the website (Featured, Charts, Search, Add Song, Login/User page). It is very responsive, so when the screen is narrower than a set treshold it transforms into a dropdown menu accessible by pressing the icon in the top right corner.

Main page

The main page is accessible at any time by pressing the logo on the navbar. It contains a list of the newest translations/songs available on the website in the FEATURED section. Each news has a title, an author and a picture of the album from where the song originates.

Under the FEATURED section the user can use the search bar to search songs that are available on the website. The search bar has autocomplete and oferes sugestions based on string prefix matching. The user can choose one of the suggestions by using the keyboard arrows and pressing enter to make a selection. Upon pressing the "Submit" button, the user will be sent to the search page where the results of the search will be listed and the user can make more searches.

Under the search bar in sections "CHARTS" and "COMMUNITY" a user can view useful statistics about the site activity. In the "CHARTS" sections, by selecting a criteria from the dropdown button from the left, can learn what are the most viewed, commented, liked songs from the webpage. The "COMMUNITY" sections will contain a table with the most active page contributions. It will contain the accounts that added the most translations.

Search page

The search page is accessible at any time by pressing the button from the navbar of by using the search bar on the main page. It will contain search results and enable the user to move to the wanted song page.

Song page

The song page is one of the most complex pages from the website. To access a certain song page, a user can click on any link on the main page from "FEATURES" and "CHARTS" or by using the search bar from the main page or the search page.

It is composed of the following sections:

  • Song header
  • Song Lyrics
  • About
  • Comment section

Song header

The song header contains the album image from where the song is from, the title, the singer/band and a short description. On the separate section in the right there is information such as the user who posted the translation, the date when the translation was added, the number of views, comments, likes. The language of the song can be changed by using the dropdown button. Upon selecting another language the lyrics, description and other elements on the page will be changed to the desired language. A user can add a translation by pressing the "Add translation" button which will transfer the user to the add translation page. A website visitor can share the translation by pressing the Tumblr and Wordpress icons.

Song Lyrics

The user can make addnotations by selecting a portion of text.

About

In the about section, the user can read the full description of the song. Moreover, if the song is on youtube, the visitor can play the song and watch the music video without leaving the website

Comment sections

In the comments section, the user can write personal opinions about the song or the translations or read other people comments

Add song page

The add song page can be accessed from the navbar. It enables the user to add a new song by writing the title, author, lyrics, description an providing a image and a youtube link

Add translation page

The add traslation page is available from the song page. It is very similar to the add song page but the user can only add the translated lyrics and description.

Login page

The login page is available from the navbar. If the user does not have an account he can create one by using the register now form.

Profile page

The profile page is available from the navbar after the user is logged in. It contains the users profile picture, username and password and enables the user to change all of them. Moreover it shows some users stats such as the number of translations, addnotations, comments and songs added.

Other pages

This refers to less frequent visited page such as "About", "Help", "Contact"
